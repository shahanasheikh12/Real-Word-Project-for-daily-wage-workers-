# DailyWork — AI Microservice (Phase 5 Scaffold)
# Python FastAPI microservice for AI/ML features.
#
# Planned endpoints:
#   POST /match-workers      → Ranked worker list for a job (Smart Matching)
#   POST /suggest-wage       → Fair wage range for a skill+location (Dynamic Wage)
#   POST /check-fraud        → Fraud score for a job post (NLP classifier)
#   POST /infer-skills       → Suggested skills for a worker (Skill Inference)
#   POST /demand-forecast    → Trending skills in an area (Time-series model)
#   POST /voice-onboard      → Audio → profile fields (Whisper speech-to-text)
#
# This folder will be implemented in Phase 5.
# Tech stack: Python 3.11+, FastAPI, scikit-learn, transformers, openai-whisper
